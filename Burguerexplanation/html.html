Burguer restaurant website proposal


Process

This project is the proposal that I presented for a website restaurant. The proposall is a responsive website version of the landing page that has been coded with Bootstrap.

My role in the project

I led the design and coding of the project, however, this project has not been published because the customer did not like my design proposal.
The website first page was designed on November 2019.


Solution:

The solution for this project that I designed have in mind the pop-art and the rock&roll culture. It wants to transmit a feeling of fun, young and attractive design.
That I learnt for this project, that sadly did not carry on, was to used efficiently the Bootstrap both for create website and the mobile version of the project.
